## 🚀 立即行动

### 第一步：安装和构建

```bash
# 1. 进入项目目录
cd D:/WorkBench/ldesign/libraries/3d-viewer

# 2. 安装 @ldesign/builder
pnpm add @ldesign/builder -D -w

# 3. 构建所有包
pnpm run build:all

# 4. 验证构建产物
pnpm run verify
```

### 第二步：运行示例

```bash
# Vue 示例
pnpm run dev:vue

# React 示例
pnpm run dev:react

# Lit 示例
pnpm run dev:lit
```

### 第三步：补充测试

```bash
# 运行测试
pnpm run test

# 生成覆盖率报告
pnpm run test:coverage
```
