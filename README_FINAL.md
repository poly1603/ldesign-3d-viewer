## 🎯 立即使用

### 安装

```bash
npm install @panorama-viewer/core three
```

### 使用

```typescript
import { PanoramaViewer } from '@panorama-viewer/core';

const viewer = new PanoramaViewer({
  container: document.getElementById('viewer'),
  image: 'panorama.jpg',
});
```
